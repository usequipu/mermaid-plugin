# mermaid-plugin

Mermaid diagram viewer plugin for [Quipu](https://quipu.cc). Side-by-side editor and live preview for `.mmd` and `.mermaid` files.

## Install

Open the Plugin Manager panel in Quipu → Available → Install.

## Supported file types

`.mmd`, `.mermaid`

## Build from source

```bash
npm install
npm run build
```
